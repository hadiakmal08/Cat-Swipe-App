# Cat-Swipe-App
# 🐾 Paws & Preferences

A fun, Tinder-style cat swiping game built with vanilla HTML, CSS, and JavaScript.

## 🐱 How to Play

Swipe through 15 randomly fetched cats and rate them:

- 👈 **Swipe Left** — Not your vibe
- 👉 **Swipe Right** — You love it!
- 💎 **Crush** — Super fave! (3 per game)
- ⏪ **Rewind** — Changed your mind? Take it back (2 per game)

At the end, see all your favourites in a results screen — crushed cats get the spotlight!

## ✨ Features

- Live cat images fetched from [The Cat API](https://thecatapi.com)
- Smooth swipe gestures (touch + mouse)
- Animated background with waves and floating particles
- Sound effects via Web Audio API
- Fully responsive — works great on mobile and desktop
- Crush spotlight, confetti, flying emojis, and more
